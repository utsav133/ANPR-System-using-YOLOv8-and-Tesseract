# Automatic Number Plate Recognition (ANPR) System

## Overview

This project implements an Automatic Number Plate Recognition (ANPR) system using YOLOv8, OpenCV, and Tesseract OCR.

The system detects vehicle license plates from images and videos, extracts the registration number, and stores the detected information in a CSV file.

## Features

* Custom YOLOv8 License Plate Detector
* Optical Character Recognition (OCR) using Tesseract
* Image-based ANPR
* Video-based ANPR
* CSV Logging
* Confidence Score Tracking
* Automatic Plate Cropping and Processing

## Tech Stack

* Python
* YOLOv8
* OpenCV
* Tesseract OCR
* NumPy
